# Linux 🐧

Libro interattivo della collana di **informatica**: usare Linux dal terminale, da zero. Dal primo comando allo
scripting bash per automatizzare il lavoro.

Stesso stile e motore degli altri libri della collana: spiegazioni discorsive, ogni comando smontato pezzo per
pezzo, un widget interattivo per capitolo, laboratori guidati con l'output atteso, riquadri colorati dal
significato fisso, esercizi con soluzioni. Contenuto 100% originale.

Apri `index.html` per la copertina con l'indice. Serve accesso a un sistema Linux (o WSL su Windows, o il
terminale di macOS, molto simile); spiegato al Cap. 1.
