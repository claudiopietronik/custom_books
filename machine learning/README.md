# Machine Learning ŷ

Libro interattivo della collana **Dati, segnali & Machine Learning**: tutto il machine learning "classico" (fuori
dalle reti neurali), dal punto di vista matematico, spiegato da zero e passo dopo passo. Impianto ispirato a *The
Elements of Statistical Learning*.

Stile: spiegazioni estremamente semplici e discorsive, **ogni formula smontata simbolo per simbolo entrando nel suo
significato** (mai una formula "nuda"), un widget interattivo per capitolo, laboratori con **scikit-learn** (codice
e output atteso), riquadri colorati dal significato fisso, esercizi con soluzioni. Contenuto 100% originale.

Apri `index.html` per la copertina con l'indice. Aiutano **Statistica** e un po' di **Algebra lineare**; i laboratori
usano Python/scikit-learn.
