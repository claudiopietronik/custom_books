# Backend Development ⚙

Libro interattivo della collana di **informatica**: come si costruisce la parte server del web, dal primo
endpoint FastAPI fino a un'applicazione completa online, dietro nginx e HTTPS.

Stesso stile e motore degli altri libri della collana: spiegazioni discorsive, ogni comando e riga di codice
smontati pezzo per pezzo, un widget interattivo per capitolo, laboratori guidati con l'output atteso, riquadri
colorati dal significato fisso, esercizi con soluzioni. Contenuto 100% originale.

Apri `index.html` per la copertina con l'indice. Prerequisiti utili: il libro **Python**; aiutano **Reti e
Internet** e **Il Web**.
