# Python per la Data Science 📊

Companion pratico del libro **Python**: l'uso delle librerie del data scientist —
NumPy, pandas, Matplotlib, seaborn, SciPy, scikit-learn, PyTorch. Taglio pratico
(uso delle librerie e workflow reali); la teoria dei modelli sta nei libri
**Machine Learning** e **Deep Learning**.

Libro statico HTML della collana `custom_books`, categoria *Dati, segnali & Machine Learning*.
Motore condiviso in `assets/` (toc.js, book.js, plot.js, style.css). Tema smeraldo (`pds-theme`).
