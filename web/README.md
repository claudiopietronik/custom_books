# Il Web — HTML, CSS e JavaScript

Libro della collana di **programmazione**, in italiano: la parte visibile del web. HTML (struttura),
CSS (aspetto) e JavaScript (comportamento), da zero fino a una piccola app interattiva. Spiegazioni
discorsive, ogni riga smontata pezzo per pezzo, e widget che girano dal vivo nel browser. Non serve
installare nulla: basta un editor di testo e un browser.

Contenuto originale scritto ex novo.

**Apri [`index.html`](index.html) con un doppio clic.** Non serve alcun server.
