# Git & GitHub 🔀

Libro interattivo della collana di **programmazione**: il controllo di versione con **Git** e la collaborazione
con **GitHub**, da zero. Dalla prima commit al workflow con pull request.

Stesso stile e motore degli altri libri della collana: spiegazioni discorsive, ogni comando smontato pezzo per
pezzo, un widget interattivo per capitolo, laboratori guidati con l'output atteso, riquadri colorati dal
significato fisso, esercizi con soluzioni. Contenuto 100% originale.

Apri `index.html` per la copertina con l'indice. Serve solo un terminale e Git installato (spiegato al Cap. 2);
nessun linguaggio di programmazione richiesto.
